Mosho

A new presentation framework

So what's it do?

If you're familiar with impress.js, you should feel pretty comfortable
(Mosho was based on it).

Keep going to find out more.

3D transform goodness

It certainly looks fancy, but doing it is easy:

<div class="mosho-slide"
     data-y=-800 data-z=800
     data-rotx=45>
   <h1>Hello, Mosho!</h1>
</div>

...in a scene tree

<div class="mosho-group" data-x=1000>
  <div class="mosho-slide" data-y=-500>
     <h1>one</h1>
  </div>
  <div class="mosho-slide" data-y=500>
     <h1>two</h1>
  </div>
</div>

Per-"slide" transitions

<div class="mosho-slide"
     data-transition="2s ease">
   <h2>That was ease-y</h2>
</div>

Non-linear linking

Would you like to know more? Yes

It's not too tricky

<div class="mosho-slide" id="slide-a"
     data-next="slide-b">
   <a href="#slide-a-more">detour</a>
</div>
<div class="mosho-slide" id="slide-a-more"
     data-next="slide-a">
   <p>scenic route</p>
</div>
<div class="mosho-slide" id="slide-b"
     data-prev="slide-a">
   <p>final destination</p>
</div>

A handy API?

mosho.enter("mySlide", function () {
  console.log("yay!")
})

...and plugins!

mosho.plugin({
  name: "scaleActiveSlide",
  preShow: function (evt) {
    prev = evt.detail.prevSlide
    next = evt.detail.nextSlide
    if (prev) { prev.scale(0.25) }
    if (next) { next.scale(4) }
  }
})
